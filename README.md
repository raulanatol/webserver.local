
# webserver.local
A simple express local server with client-side routing

## Install

```console
$ npm install -g webserver.local
```

## Usage

```console
$ webserver.local
```

Parameters:

| Parameter       | Description         |
| ----------------|---------------------|
| -p, --port      | Port to open        |
| -d, --directory | The index.html path | 
| -t, --trace     | Trace mode          |
